# Порядок выполнения задач MVP (v6.13)

Источник оценок и порядка: `planning/mvp/current/gantt/mvp_gantt_chart_current.puml` (Gantt v6.13, rescoped).

> Примечание по планированию Q2: **Фаза 1 = весь MVP**. "Фаза 2" (прочие Q2 задачи, не относящиеся к MVP) в этом списке не описана.

## Фаза 0. Аналитика (вся вперед)

- `AN_AP1` Аналитика процесса согласования
- `AN_APR1` Аналитика страницы "Согласования"
- `AN_PKG1` Аналитика страницы "Пакеты"
- `AN_C1` Аналитика списка внедрений
- `AN_P1` Аналитика адаптации пилотов
- `AN_CHD1` Аналитика деталки внедрения
- `AN_CHF1` Аналитика формы внедрения
- `AN_PA1` Аналитика блоков "Артефакты" и "Связанные сущности" на страницах пилота
- `AN_SC1` Аналитика списка скоркарт
- `AN_SCD1` Аналитика деталки скоркарты
- `AN_SCF1` Аналитика формы скоркарты
- `AN_CD1` Аналитика источников lineage ("создано из")

## Блок 1. Approval/Ratification + Packages (foundation)

- `BE_AP1` Backend процесс согласования (`ApprovalInstance/Stage/Assignment/Decision`)
- `BE_INT1` Интеграция ЖЦ (Pilot/Deployment) с результатами согласования
- `BE_PL1` ЖЦ пилота
- `BE_ART1` Артефакты (ссылки на документы)
- `BE_SC_INT1` Интеграция скоркарт (hook-места) (5 дней)
- `BE_NOT1` Уведомления
- `BE_APR1` Backend страницы "Согласования"
- `BE_PKG1` Backend страницы "Пакеты"
- `FE_APR1` Frontend страницы "Согласования" (10 дней)
- `FE_PKG1` Frontend страницы "Пакеты" (10 дней)

## Блок 2. Внедрения (Deployment)

- `BE_CL1` ЖЦ внедрения (submit_for_approval/deploy/rollback) (5 дней)
- `BE_CHD1` Backend деталки внедрения (5 дней)
- `BE_CHF1` Backend формы внедрения (5 дней)
- `BE_C1` Backend списка внедрений (5 дней)
- `FE_C1` Frontend списка внедрений (5 дней)
- `FE_CHD1` Frontend деталки внедрения (5 дней)
- `FE_CHF1` Frontend формы создания/редактирования внедрения (5 дней)

## Блок 3. Scorecards

- `BE_SC1` Backend списка скоркарт (10 дней; включает шаблоны скоркарт)
- `BE_SCD1` Backend деталки скоркарты (5 дней)
- `BE_SCF1` Backend формы создания/редактирования скоркарты (4 дня)
- `FE_SC1` Frontend списка скоркарт (10 дней)
- `FE_SCD1` Frontend деталки скоркарты (8 дней)
- `FE_SCF1` Frontend формы создания/редактирования скоркарты (10 дней)

## Блок 4. Pilots (страницы/формы)

- `BE_P1` Backend списка пилотов
- `BE_PA1` Backend блоков "Артефакты" и "Связанные сущности" на пилоте
- `FE_P1` Frontend списка пилотов
- `FE_PD1` Frontend деталки пилота
- `FE_PA1` Frontend блоков "Артефакты" и "Связанные сущности" на пилоте (create/edit/detail)

## Блок 4.5. Simulations (страницы/формы, только блоки)

- `FE_SA1` Frontend блоков "Артефакты" и "Связанные сущности" на симуляции (detail/create/edit)

## Блок 5. Источники created_from (preview в формах)

- `BE_CD1` Backend источников (created_from): автосоздание + preview endpoint для форм
- `FE_CD1` Frontend preview источников (created_from) в формах

## Финал (общесистемное + QA MVP)
