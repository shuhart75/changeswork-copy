# Заметки к MVP-задаче — Артефакты (core) (BE + API + права)

Этот файл сопровождает MVP-задачи по блоку «Артефакты».

## Объём
Artifacts core должен поддерживать прикрепление артефактов к:
- PILOT
- SIMULATION
- DEPLOYMENT

В MVP артефакты — это ссылки на внешние документы (без загрузки файлов).

## Передача требований
См.: `final-spec/REQ_artifacts_core.md`

## Критерии приемки (кратко)
- CRUD работает для всех 3 типов сущностей.
- Права наследуются от родительской сущности:
  - viewer может просматривать список
  - editor может создавать/редактировать/удалять
- Базовая серверная валидация `url/title`.
