# Handoff Bundle (final-spec)

Эта папка предназначена для передачи команде и демонстрации заказчику.

## Что внутри

- `prototype.html` (MVP) интерактивный HTML прототип
- `spec_domain_model.md` доменная модель (источник правды)
- `spec_data_model.puml` ER-диаграмма
- `spec_state_machine.puml` диаграммы состояний
- `REQ_scorecards_frontend.md` требования по Frontend для скоркарт
- `REQ_scorecards_backend.md` требования по Backend API для скоркарт
- `CLAUDE.md` рабочее руководство/контекст (исторически в проекте используется как "guide")
- `mvp_gantt.puml` актуальная диаграмма Ганта MVP (planning v6.15, источник оценок: ProjectLibre Q2.pod)
- `mvp_tasks_summary.md` индекс задач MVP
- `mvp_tasks_execution_order.md` порядок выполнения задач MVP
- `mvp_tasks_projectlibre_alignment.md` описание скоупа по задачам (AN+DEV), выровненное с `Q2.pod`
- `sources/projectlibre/Q2.pod` исходный файл ProjectLibre
- `sources/projectlibre/Q2.json` экспорт из `.pod` (MPXJ), удобен для парсинга

## Как открыть прототип

Откройте `prototype.html` в браузере (локально, без сборки).
