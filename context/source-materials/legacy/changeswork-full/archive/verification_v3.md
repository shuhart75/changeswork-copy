# Верификация mvp_gantt_chart_v3.puml - Пошаговая проверка загрузки ресурсов

## Методология проверки

Для каждого ресурса прослеживаем все задачи в хронологическом порядке, проверяя:
1. Нет ли наложений (одновременного выполнения задач)
2. Корректность зависимостей
3. Соответствие загрузки возможностям ресурса
