# Delivery Prototype Notes

Срез: `bt-publication`  
Дата обновления: `2026-04-27`

Покрытие прототипа:
- подходящий и неподходящий контекст для кнопки `Сформировать БТ`;
- локальная вставка черновика запроса по данным симуляции;
- свободное редактирование черновика пользователем;
- отсутствие автоотправки после вставки;
- показ и обновление `btUrl` после успешной публикации.

Что важно для handoff:
- черновик строится на фронте из имени пользователя, риск-параметров и бизнес-цели;
- при неподходящем контексте окно агента не исчезает, меняется только доступность действия по БТ;
- `btUrl` показывается и в окне, и в данных симуляции.
