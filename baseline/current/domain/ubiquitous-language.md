# Ubiquitous Language

Дата обновления: `2026-04-23`
Source: `context/source-materials/current-system/diagrams/raw/spec_domain_model.md`

## Основные термины
- **Инициатива** (`Initiative`) — верхнеуровневая группировка стратегического изменения.
- **Симуляция** (`Simulation`) — анализ исторических данных и источник гипотез/исходных данных.
- **Пилот** (`Pilot`) — контролируемый эксперимент с версионированием через `PilotVersion`.
- **Внедрение** (`Deployment`) — продуктовое внедрение с версионированием через `DeploymentVersion`.
- **Скоркарта** (`Scorecard`) — агрегат конфигурации и финансовых эффектов с собственным versioning.
- **Шаблон скоркарты** (`ScorecardTemplateVersion`) — versioned-описание полей, default values и порогов критичности.
- **Lineage** — ограниченная MVP-трассировка происхождения через `Deployment.lineage_simulation_id`.
- **Пакет** (`Package`) — рабочая группировка нескольких `ApprovalInstance` из `awaiting_ratification` для совместного ratification.
- **Approval** — согласование риск-менеджерами по 0..* этапам.
- **Ratification** — финальное утверждение руководством, 1 этап в MVP.

## Термины ролевой модели
- **ПРМ** (`prm`) — управляет инициативами своего продукта, может работать со скоркартами своего продукта, отправлять версии на согласование и формировать пакеты своего scope.
- **Методолог** (`methodologist`) — управляет скоркартами и документами своего продукта; на странице `Пакеты` имеет доменное исключение и может собирать смешанные пакеты по разным продуктам.
- **Approver** — участник approval-этапов, видит свои назначения.
- **Ratifier** — участник ratification-этапа, видит свои назначения.
- **Админ** (`admin`) — полный доступ.

## Термины жизненного цикла
- **current version** — текущая отображаемая версия логической сущности `Pilot` или `Deployment`.
- **production version** — текущая продуктивная версия `Deployment`.
- **awaiting_ratification** — очередь ручного утверждения после approval.
- **approval_cancelled / ratification_cancelled** — отзыв версии из процесса согласования.
- **archive** у `Scorecard` — статус отсутствия связей использования; не равен архиву доменных версий.
