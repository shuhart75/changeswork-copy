# Context — Initiatives

## Purpose
Верхнеуровневая рамка, объединяющая все доменные сущности одного стратегического изменения.

## Main objects
- `Initiative`

## Key rules
- Инициатива принадлежит одному продукту.
- Внутри инициативы могут существовать многие `Simulation`, `Pilot`, `Deployment`.
- Статус инициативы считается автоматически из статусов вложенных сущностей.
- Архивирование инициативы не каскадирует архивирование вложенных сущностей.
