# Context — Product Catalog

## Purpose
Классификация продуктового ландшафта, к которому привязываются инициативы и ролевой scope.

## Main objects
- `Product`
- `SubProduct`

## Key rules
- Инициатива принадлежит одному `Product`.
- Инициатива может быть связана с несколькими `SubProduct`; если связь не задана, трактуется как весь продукт.
- В текущем MVP подпродукты не используются в `Scorecard`.
